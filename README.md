# EventRent

## Deskripsi Aplikasi
EventRent adalah aplikasi penyewaan perlengkapan acara yang dibuat menggunakan Django. Aplikasi ini memudahkan pengguna untuk melihat barang, menambahkan barang ke keranjang, melakukan penyewaan, serta melihat laporan penyewaan.

## Cara Instalasi

Clone repository

```bash
git clone https://github.com/NAMA-AKUN-GITHUB/eventrent-kelompok6.git
```

Masuk ke folder project

```bash
cd eventrent-kelompok6
```

Install library

```bash
pip install -r requirements.txt
```

Migrasi database

```bash
python manage.py migrate
```

Jalankan server

```bash
python manage.py runserver
```

Buka browser

```
http://127.0.0.1:8000/
```
